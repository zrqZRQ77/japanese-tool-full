# Design Reference Sources

`frontend/design-reference.html` is a visual comparison page, not a document reader.

It contains three visual panels:

1. `我的网站`: the current Yomeru / Dokedo visual system used by `frontend/design-system.css`.
2. `Claude 风格`: a local visual interpretation of the Claude-like design direction, focused on warm paper surfaces, low visual noise, calm typography, and clear primary action.
3. `Material 3`: a local visual interpretation of Material Design 3 roles, including primary, primary container, surface, surface container, outline, and common button types.

Reference links:

- Claude design.md: https://getdesign.md/claude/design-md
- Material Design 3: https://m3.material.io/
- Local Claude-style visual rules: `CLAUDE_VISUAL_DESIGN.md`
- Project design rules: `DESIGN.md`
- Active project design CSS: `frontend/design-system.css`

The HTML page should stay visual-first. Long explanations and source links belong here or in `DESIGN.md`, not in the main comparison panels.
