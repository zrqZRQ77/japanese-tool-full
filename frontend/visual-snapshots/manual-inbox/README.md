# Manual Screenshot Inbox

把用户手动截的最新页面截图放在这里。

命名格式：

```text
版本号-页面名.png
```

示例：

```text
20260704-19-home.png
20260704-19-reading.png
20260704-19-vocab.png
20260704-19-grammar.png
```

常用页面名：

- `home`
- `reading`
- `vocab`
- `grammar`
- `practice`
- `discover`
- `test`
- `history`
- `settings`

小改动只需要放受影响页面。稳定大版本才需要完整页面组。
