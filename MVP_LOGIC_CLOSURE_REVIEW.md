## 2. MVP 核心闭环定义

本项目的主闭环应当是：

> 获取日语文章 → 生成阅读材料 → 点击词语查看详情 → 收藏生词 → 复习或练习 → 查看学习进度 → 找下一篇文章

对于 MVP，不要求每一个辅助功能都很强，但要求：

- 用户知道从哪里开始。
- 每个主要动作都有明确结果。
- 完成后知道下一步去哪里。
- 没有数据时有返回主流程的入口。
- 失败时有替代路径。
- 页面上展示出来的能力，不能没有实现或没有后续入口。
