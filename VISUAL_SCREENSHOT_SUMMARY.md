### 当时看到的问题

- 首页 Logo 外面有明显边框，用户希望改成更轻的圆形或直接无边框。
- Logo 旁边不需要再显示文字，主页标题已经承担品牌识别。
- 输入框和功能卡背后的局部背景与页面左右背景不一致，看起来像多了一层底色。
- 输入框下方几个文字按钮 hover/active 背景和按钮本身颜色太接近，不够协调。
- 功能卡被选中时，卡片内部演示区也是白色小卡片，形成“卡片里面还有卡片”的分层感。

### 已确认方向

- 首页首屏只聚焦：
  - 左上角图形 Logo
  - `読める · 日语阅读辅助`
  - 大输入框
  - 上传文件 / 导入链接 / 使用示例
  - 开始阅读
- Logo 不和标题放在一起，不在 Logo 旁边重复文字。
- 首页背景应整体统一，不能在内容区域后方出现一块额外底色。
- 输入框是首屏唯一明显框起来的核心区域。
- 功能卡下滑后展示，两列等高，卡片内部不要产生明显二次卡片感。

### 参考截图

### 当时看到的问题

- 旧 Logo 使用了汉字 `読`，和整体新 Logo 不统一。
- 设置与帮助图标也曾使用汉字，需要统一为图形图标。
- 正文区域出现大量词语背景色，影响阅读；用户希望默认正文干净，点击词语或语法时再展示标注/详情。
- 阅读页曾有剩余绿色色系，需要改成紫色体系或中性色。
- 中间阅读区域外面有一个大卡片框，里面又包含工具栏、正文、底部练习卡，显得层级过重。
- 用户希望工具栏、正文、底部“进入练习”区域分开即可，不需要再被一个大外框包住。
- 右侧详情面板在点击词语后显示即可，默认状态应清爽。
- 某次修复后左侧导航文字竖排、右侧卡片错乱，属于 HTML 结构和 CSS 覆盖不同步导致。

### 已确认方向

- 阅读页左侧导航保留，但 Logo 只显示最新图形，不带文字。
- 阅读正文默认是深灰/近黑文字，不使用品牌紫作为正文色。
- 词语背景默认不大面积出现；点击词语、语法、工具状态时再展示浅紫/浅色提示。
- 中间区域应拆为三个独立层：
  - 工具栏
  - 阅读正文面板
  - 阅读完成后的练习入口
- 不再用一个大卡片包住这三块。
- 阅读页整体背景不应偏粉紫，优先使用中性微灰或白色纸面。
- 紫色只用于当前工具、焦点、标签和主动作。

### 参考截图

- `/Users/zhouruoqi/Documents/截屏2026-07-03 18.39.11.png`
- `/Users/zhouruoqi/Documents/截屏2026-07-03 20.51.29.png`

### 当时看到的问题

- 顶部筛选按钮颜色区分不明显，尤其是“全部状态”这类控件。
- 词汇主文字大面积使用紫色，用户认为深灰或近黑会更清晰。
- 旧绿色状态色残留，需要统一到最新设计系统。
- 生词本页面有“大卡片包小卡片”的问题，看起来臃肿。
- 列表、筛选、表头、词条应该属于同一个工作流，而不是多层卡片堆叠。

### 已确认方向

- 生词本是“列表工作台”，不是卡片集合。
- 搜索、筛选、表头、列表在同一块平面上组织。
- 外层大卡片应弱化或透明。
- 词汇主文字使用深灰/近黑，提高可读性。
- 紫色只用于导航选中、主操作、JLPT 小标签、焦点状态。
- 状态点规则：
  - 掌握/认识：紫色或品牌强调色
  - 待复习/有点犹豫：黄色
  - 没记住：红色
- 用户明确确认“有点犹豫、待复习”改成黄色更好。

## 4. 语法本

### 参考截图

- `/Users/zhouruoqi/Documents/截屏2026-07-03 18.40.04.png`
- `/Users/zhouruoqi/Documents/截屏2026-07-03 23.03.46.png`

### 当时看到的问题

- 页面整体布局和功能看起来乱。
- 添加语法、搜索语法、热门查阅等模块堆在一起，视觉优先级不清楚。
- 大卡片套小卡片的问题明显。
- 右侧“我的语法收藏”空状态像卡片里的卡片，显得重。

### 已确认方向

- 语法本应成为一个清晰的语法查询工作台。
- 主任务优先是“查语法”，添加/收藏属于次级流程。
- 页面应减少外层大卡片。
- 查语法、热门查阅、我的收藏应有清楚分区，但不要多层边框嵌套。
- 空状态应使用轻提示、浅底或虚线区域，而不是再套一张重卡片。

## 5. 练习、资料、学习历史、设置与帮助

### 相关讨论

- 用户要求按照主页和阅读页的新逻辑，把“练习、资料、学习历史、设置与帮助”等页面也更新。
- 后续又指出这些页面也存在大卡片套小卡片的情况。

### 已确认方向

- 这些页面归入二级工作区。
- 不做营销式页面，也不做复杂装饰。
- 外层页面只负责宽度、留白和背景，不画明显卡片边框。
- 内部真正独立的对象才使用卡片。
- 列表、图表、表单、状态区要靠分割线、标题和留白组织。
- 设置与帮助图标不再使用汉字，统一使用图形图标。

   - 首页
   - 阅读页
