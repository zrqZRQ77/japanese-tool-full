# Yomeru 中文词库人工审核操作指南

## 最简单的启动方式（Mac）

在 Finder 中打开：

`/Users/zhouruoqi/Downloads/japan/japaneselearning-vocab/scripts/`

双击：

`start-offline-chinese-human-review.command`

终端会自动启动本地审核服务器，并打开浏览器页面：

`http://127.0.0.1:4175`

如果 macOS 第一次阻止打开，可右键该文件，选择“打开”。

## 终端启动方式

```bash
cd /Users/zhouruoqi/Downloads/japan/japaneselearning-vocab
node scripts/build-offline-chinese-human-review-packet.mjs
node scripts/offline-chinese-human-review-server.mjs
```

审核期间不要关闭运行服务器的终端窗口。审核结果每次点击后都会自动写入：

`audits/offline-chinese-coverage/20260723/human-approval-packet.json`

## 页面操作

1. 在顶部填写审核人姓名。
2. 查看日文表记、假名读音、中文草稿、置信度和证据摘要。
3. 按实际判断选择：
   - **通过并下一条**：中文草稿可以直接使用。
   - **修改后通过**：先修改“最终中文释义”，并填写修改理由。
   - **拒绝**：填写拒绝理由，该词条不会进入正式中文词库。
4. 保存后页面自动进入下一条待审核项目。
5. 可使用筛选器查看待审核、已通过、修改后通过或已拒绝项目。

## 判断标准

只有同时满足以下条件才选择“通过”：

- 日文表记与读音对应正确。
- 中文释义自然、简洁，没有扩大原义。
- 不会覆盖同音词或同形异读词。
- 词性与语义范围合理。
- 页面列出的证据足以支持该释义。

不确定时不要勉强通过：

- 可以选择“修改后通过”并缩小释义范围。
- 证据或语义明显不可靠时选择“拒绝”。

## 安全说明

- 审核页面仅监听本机 `127.0.0.1`，外部网络无法访问。
- 不上传阅读内容或审核数据。
- 不使用数据库。
- 不会自动批准词条。
- 重新生成审核包时，只要词条和中文草稿没有变化，已完成的人工决定会被保留。
