# Yomeru 离线中文词库：数据源与许可证方案

Updated: 2026-07-23

> 本文是工程数据接入规范，不构成法律意见。每次生产接入前应重新确认来源条款和数据版本。

## 结论

正式中文释义继续以 Yomeru 人工审核数据为唯一发布层。其他来源只承担词形、英文释义、实体消歧、用法例句或候选证据，不把不同许可证的原文混入一个无法追踪来源的数据文件。

## 数据分层

### Layer A：Yomeru 人工中文释义

用途：正式中文释义、人工修订、运行时最高优先级。

要求：

- 每条释义记录审核状态和来源证据。
- 禁止复制来源不明或商业词典的定义。
- AI 只能生成待审核草稿，不得自动发布。
- 专有名词、歧义词和上下文词必须人工处理。

### Layer B：JMdict / EDRDG

许可：CC BY-SA 4.0。

允许：

- 日语词形、读音、词性和英文释义回退。
- 作为中文候选的人工审核证据。
- 商业产品使用。

必须：

- 在 About、Sources 或等效页面说明 JMdict/EDRDG。
- 提供本地许可文件或官方许可链接。
- 显著提取、词表或基于其形成的材料必须明确致谢。
- 不得对 JMdict 原始或衍生材料主张独占版权。
- JMdict 数据与项目人工中文释义保持可识别分层。
- 生产数据保持定期更新；当前项目规则为至少每月检查一次。

### Layer C：Wikidata 结构化数据

许可：main、Property、Lexeme、EntitySchema 等结构化命名空间采用 CC0。

允许：

- Q/L 实体 ID。
- 标签、别名和语言代码。
- 专有名词消歧和结构化事实。

要求：

- 记录实体 ID、抓取日期和字段来源。
- 建议在 Sources 中说明使用 Wikidata。
- 不将帮助页、讨论页或其他命名空间的说明文字误当作 CC0 数据。
- 只作事实和实体层，不直接替代正式中文词义。

### Layer D：Wiktionary / Wikimedia 文字

许可：通常为 CC BY-SA 4.0，并可能同时适用 GFDL；具体页面可能有额外要求。

用途：仅作人工审核证据，例如词源、用法、词形和歧义判断。

要求：

- 保存页面 URL、版本、历史链接和修改说明。
- 满足署名和相同方式共享要求。
- 与项目人工中文发布层隔离保存。
- 默认不复制页面定义正文到正式词库。
- 接入前逐页核验页脚、历史和额外署名说明。

### Layer E：Tatoeba 例句

许可：文本导出默认 CC BY 2.0 FR，部分句子另有 CC0 导出；音频许可按贡献者分别决定。

用途：

- 词义歧义和搭配的人工审核语境。
- 后续独立例句数据集。

要求：

- 优先使用明确 CC0 的句子。
- CC BY 句子必须保存句子 ID、作者和许可。
- 例句和词典定义分开存储。
- 不把翻译句直接视为正式词义。
- 不使用许可不明的音频。

### Layer F：Yomeru 项目语料

许可：项目自有数据。

用途：确认审核目标的完整日文表记、读音、基本形、词性和项目测试语境。语料不能单独证明正式中文词义；P3 复合词必须同时关联 JMdict 组件证据或许可明确的外部语义证据。

### Layer G：日本政府 PDL1.0 内容

许可：公共数据利用规约第 1.0 版（PDL1.0），与 CC BY 4.0 兼容；使用前仍需逐页确认机关利用条款、第三方权利和个别例外。

用途：官方行政、社会保障和政策术语的隔离审核证据。必须保存发布机关、页面 URL、利用条款 URL、抓取日期及加工说明；不直接复制长段正文到正式词库。

## 禁止来源

以下内容不得进入候选或正式数据：

- 未明确授权的商业词典。
- 搜索引擎摘要、聚合网站或来源不明的 API 返回。
- 抓取后通过轻微改写规避来源的定义。
- 许可证不允许商业再分发的数据。
- AI 自动翻译但没有独立证据和人工审核的释义。

## 发布门禁

一条中文释义只有同时满足以下条件，才允许写入正式中文数据：

1. 至少有一个可追溯的独立证据来源。
2. 许可证允许当前用途，或只引用事实而未复制受保护文字。
3. 中文释义由人工确认自然、准确且不包含来源专有表达。
4. 歧义、专有名词和上下文依赖风险已经处理。
5. 审核状态、审核人和时间完整。
6. 数据格式、重复、空释义和冲突检测通过。

## 接入顺序

1. B3 先建立人工审核队列，不生成正式中文释义。
2. 首批审核优先处理高频、JLPT 和真实阅读未命中词。
3. 电车挑战合并并发布后，中文词库分支同步最新 main。
4. 只有完成许可证与人工审核门禁后，才接入首批 500–1000 词。
5. 运行时接入使用独立 Preview、PR 和 Production 流程。
